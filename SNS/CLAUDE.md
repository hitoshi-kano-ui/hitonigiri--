## Architecture

### ディレクトリ構造

```

├── README.md

```

## ルール

1. **投稿記録** → 投稿内容と日時を管理する
2. **DEVLOG.md** → 作業記録を時系列で追記
3. **README.md** → プロジェクト概要のみ（日々の詳細は書かない）
